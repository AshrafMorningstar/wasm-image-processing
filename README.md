## :floppy_disk: Installation & Usage

1. **Clone the repository:**
   ```bash
   git clone https://github.com/AshrafMorningstar/wasm-image-processing.git
   cd wasm-image-processing
   ```

2. **Install dependencies:**
    npm install

3. **Run the application:**
    npm start
